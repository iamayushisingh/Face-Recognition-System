# Face Recognition System

This project is a beginner-level implementation of a face recognition and detection system using Python and OpenCV.

## Features
- Real-time face detection using webcam
- Basic image preprocessing
- Face detection using OpenCV Haarcascade

## Technologies Used
- Python
- OpenCV
- NumPy

## Project Status
Currently improving the project by working on better face recognition accuracy and dataset handling.

## Future Improvements

- Improve preprocessing techniques
